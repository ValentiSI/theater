python manage.py runserver - запустить сервер

python manage.py makemigrations - если вы изменили модели, этой командой создается миграция - обновление базы данных. Создается код
    для перехода базы данных в новое состояние

python manage.py migrate - обновление/создание базы данных (приминили созданный ранее код)
